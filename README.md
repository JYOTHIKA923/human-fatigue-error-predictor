# human-fatigue-error-predictor
Real-time system that predicts human fatigue and error risk using behavioral signals like typing speed, pauses, and mouse movement.

#  Human Error & Fatigue Predictor

##  Overview
Human fatigue and cognitive overload are major causes of critical errors in industries like healthcare, aviation, and IT.  

This project is a **real-time fatigue detection system** that analyzes user behavior such as typing speed, pause patterns, mouse movement, and typing errors to predict the likelihood of human error.

---

## Problem Statement
Fatigue often goes unnoticed until a mistake occurs, leading to serious consequences.  
There is a need for a **non-invasive, real-time system** to detect early signs of fatigue and prevent human errors.

---

##  Solution
We built a **behavioral-based fatigue prediction system** that:
- Monitors user activity in real-time  
- Calculates a **Fatigue Score (0–10)**  
- Classifies risk into **LOW / MEDIUM / HIGH**  
- Provides actionable reasons for fatigue  

---

##  Key Features
-  Tracks typing speed, pauses, backspaces  
-  Monitors mouse movement behavior  
-  Generates **Fatigue Score (0–10)**  
-  Risk classification (LOW / MEDIUM / HIGH)  
-  provides reasons fatigue   
-  Simple and interactive GUI  

---

##  System Architecture
1. **Input Layer:** Captures user behavioral data  
2. **Processing Layer:** Analyzes patterns using rule-based logic  
3. **Scoring Engine:** Computes fatigue score  
4. **Decision Module:** Determines fatigue level  
5. **Output Layer:** Displays results via GUI    

---

##  Tech Stack
- **Language:** Python  
- **GUI:** Tkinter  
- **Logic:** Rule-based AI system    

---

##  How It Works
1. User starts typing for a fixed duration  
2. System tracks behavioral signals  
3. Data is analyzed in real-time  
4. Fatigue score is calculated  
5. Risk level is displayed with reasons
